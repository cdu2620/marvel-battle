Marvel Battle
This is a fighting video game where players can choose Marvel characters to battle each other. Currently, the characters are Captain America and Iron Man, with a single-player mode AI of Sif. Each character has a specific skillset, and they will have to battle each other until one runs out of lives first. 

The game can be run through any platform that supports Python. Pygame must be installed beforehand. 

Only the pygame library is necessary.

To run the game, on Mac press the command, shift and e key together at the same time.
